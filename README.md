# TNS Cashew Nut Quality Audit Mobile APP

## Tech Stack
 - React-native
 - TypeScript
 - Expo
 - Firebase

## Requirements
 - Expo
 - Firebase account

## Setup

1. Run
    ```
    npm install
    ```

2. Setup firebase connection database.

    ```
       copy src/core/utils/config.example.js to src/core/utils/config.js
       replace object values with your firebase configuration
    ```

3. Run the app

    ```
   expo start
